## 🚀 快速開始

### 環境需求

- Python 3.8 或更高版本
- pip(Python 套件管理器)

### 安裝步驟

1. 複製專案:
```bash
git clone https://github.com/Youchenjiang/Script-List.git
cd Script-List
```

2. 進入你想使用的腳本目錄:
```bash
cd category/script-name
```

3. 安裝依賴套件(如需要):
```bash
pip install -r requirements.txt
```

4. 執行腳本:
```bash
python script.py
```

##  使用指南

每個腳本都包含:
- **README.md**: 詳細文檔與使用說明
- **requirements.txt**: 所需 Python 套件清單
- **examples/**: 範例輸入與輸出

請閱讀各腳本的 README 了解具體使用方法。
